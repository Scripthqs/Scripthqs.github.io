# 字体标签  

 `<u>`：下划线标记

```html
<u>下划线标记</u>
```

<u>下划线标记</u>

`<s>或<del>`：中划线标记（删除线）

```html
<s>中划线标记（删除线）</s>
```

<s>中划线标记（删除线）</s>

`<i>或<em>`：斜体标记

```html
<i>斜体标记</i>
```

<i>斜体标记</i>

`<sup>和<sub>`：上标和下标

```html
上标<sup>上标</sup>
下标<sub>下标</sub>
```

上标<sup>上标</sup>

下标<sub>下标</sub>

`<i>`标签经常用来做网页的小图标。

- `<font>`:可以设置字体的颜色大小，但都使用css，**不要使用**。
- `<b>或<strong>`：可以加粗字体，同理属于css,**不要使用**。
