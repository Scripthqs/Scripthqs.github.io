#### 参数支持

|    参数名    | 参数描述 | 参数类型 | 默认值 |     可选值     |
| :----------: | :------: | :------: | :----: | :------------: |
|     icon     | 字体图标 |  string  |   无   |  set/download  |
| iconPosition | 图标方向 |  string  | 'left' | 'left'/'right' |
|   disabled   | 禁用按钮 | boolean  | false  |  true/ false   |
|   loading    | 图标加载 | boolean  | false  |  true/ false   |
